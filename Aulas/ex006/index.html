Principais formataçõe

Negrito / Destaque

Nesta frase temos um termo em negrito, usando a tag b(não semântica)

Nesta frase temos um termo em destaque, usando a tag strong(semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico, utilizando a tag i(não semântica)

Nesta frase temos um termo em ênfase, usando a tag em(semântica)

Texto Marcado

Podemos também criar um texto marcado, usando a tag mark

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto excluído

Podemos marcar um texto como excluído, para ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido, para ser lido com ênfase e indicar que ele foi indicado
